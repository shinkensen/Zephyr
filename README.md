# Zephyr
High performance ESP-NOW based Hypercar!
<br/>
Features:
- 2S 2200mah LiPo Battery
- Custom body, transmission, steering and pcb
- 4300kv Brushless Motor
- Active Aero
- Expected Max Speed 40-50MPH
